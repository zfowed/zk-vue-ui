## Button 按钮

### 基本使用

```html
/*vue*/

<template>
  <div>
    <zk-button>按钮</zk-button>
    <zk-button border>按钮</zk-button>
  </div>
</template>

<script>
  export default {}
</script>
```

### 禁用

```html
/*vue*/

<template>
  <div>
    <zk-button disabled>按钮</zk-button>
    <zk-button disabled border>按钮</zk-button>
  </div>
</template>

<script>
  export default {}
</script>
```
